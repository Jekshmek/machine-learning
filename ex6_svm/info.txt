Опорные векторные машины
В этом упражнении вы будете использовать машины опорных векторов (SVM) для создания классификатора спама.
Файлы, необходимые для этого упражнения
• ex6.mlx - MATLAB Live Script, который шаг за шагом выполняет упражнение
Часть 1
• ex6data1.mat - Пример набора данных 1
• ex6data2.mat - Пример набора данных 2
• ex6data3.mat - Пример набора данных 3
• svmTrain.m - функция обучения SVM
• svmPredict.m - функция прогнозирования SVM
• plotData.m - график 2D данных
• visualizeBoundaryLinear.m - График линейной границы
• visualizeBoundary.m - График нелинейной границы
• linearKernel.m - Линейное ядро ​​для SVM
• submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
• * gaussianKernel.m - ядро ​​Гаусса для SVM
• * dataset3Params.m - Параметры, используемые для набора данных 3
Часть 2
• spamTrain.mat - тренировочный набор для спама
• spamTest.mat - набор тестов на спам
• emailSample1.txt - Образец письма 1
• emailSample2.txt - образец электронного письма 2
• spamSample1.txt - образец спама 1
• spamSample2.txt - образец спама 2
• vocab.txt - словарный список
• getVocabList.m - Загрузить список словарей
• porterStemmer.m - функция стемминга
• readFile.m - читает файл в символьную строку
• submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
• * processEmail.m - Предварительная обработка электронной почты
• * emailFeatures.m - Извлечение функций из электронных писем
