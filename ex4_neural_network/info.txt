Обучение нейронных сетей
В этом упражнении вы внедрите алгоритм обратного распространения для нейронных сетей и примените его к задаче
распознавание рукописных цифр.

Файлы, необходимые для этого упражнения
• ex4.mlx - MATLAB Live Script, который шаг за шагом выполняет упражнение
• ex4data1.mat - обучающий набор рукописных цифр
• ex4weights.mat - Параметры нейронной сети для упражнения 4
• submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
• displayData.m - функция, помогающая визуализировать набор данных
• fmincg.m - процедура минимизации функции (аналогично fminunc)
• sigmoid.m - сигмовидная функция
• computeNumericGradient.m - численное вычисление градиентов
• checkNNGradients.m - функция, помогающая проверить ваши градиенты
• debugInitializeWeights.m - функция для инициализации весов
•Gnet.m - функция прогнозирования нейронной сети.
• * sigmoidGradient.m - вычислить градиент сигмовидной функции
• * randInitializeWeights.m - Произвольная инициализация весов
• * nnCostFunction.m - функция стоимости нейронной сети
