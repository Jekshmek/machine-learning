Кластеризация K-средних и анализ главных компонентов
В этом упражнении вы реализуете алгоритм кластеризации K-средних и применяете его для сжатия изображения. в
Во второй части вы будете использовать анализ основных компонентов, чтобы найти низкоразмерное представление изображений лиц.
Файлы, необходимые для этого упражнения
• ex7.mlx - MATLAB Live Script, который поможет вам выполнить упражнение
• ex7data1.mat - Пример набора данных для PCA
• ex7data2.mat - Пример набора данных для K-средних
• ex7faces.mat - Набор данных лиц
• bird small.png - Пример изображения
• displayData.m - отображает 2D данные, сохраненные в матрице
• drawLine.m - рисует линию над существующей фигурой
• plotDataPoints.m - Инициализация для центроидов K-средних
• plotProgresskMeans.m - отображает каждый шаг K-средних по мере прохождения
• runkMeans.m - запускает алгоритм K-средних
• submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
• * pca.m - выполнить анализ основных компонентов
• * projectData.m - Проецирует набор данных в пространство меньшего размера
• *coverData.m - восстанавливает исходные данные из проекции
• * findClosestCentroids.m - Поиск ближайших центроидов (используется в K-средних)
• * computeCentroids.m - вычислить средние значения по центроидам (используются в K-средних)
• * kMeansInitCentroids.m - Инициализация для центроидов K-средних
