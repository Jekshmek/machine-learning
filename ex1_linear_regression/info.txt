 Главные скрипты функции ex1.m и ex1_multi.m
 
 ex1.mlx - MATLAB Live Script, который шаг за шагом выполняет упражнение
 ex1data1.txt - Набор данных для линейной регрессии с одной переменной
 ex1data2.txt - Набор данных для линейной регрессии с несколькими переменными
 submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
 
 * warmUpExercise.m - Простой пример функции в MATLAB
 * plotData.m - функция для отображения набора данных
 * computeCost.m - функция для расчета стоимости линейной регрессии
 * GradientDescent.m - Функция для запуска градиентного спуска
 
 ** computeCostMulti.m - функция стоимости для нескольких переменных
 ** GradientDescentMulti.m - градиентный спуск для нескольких переменных
 ** featureNormalize.m - функция для нормализации функций
 ** normalEqn.m - функция для вычисления нормальных уравнений
