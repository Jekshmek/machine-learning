Системы обнаружения и рекомендации аномалий
В этом упражнении вы реализуете алгоритм обнаружения аномалий и применяете его для обнаружения неисправных серверов на
сеть. Во второй части вы будете использовать совместную фильтрацию для создания системы рекомендаций для фильмов.
Файлы, необходимые для этого упражнения
• ex8.mlx - MATLAB Live Script, который поможет вам выполнить упражнение
• ex8data1.mat - первый пример набора данных для обнаружения аномалий
• ex8data2.mat - второй пример набора данных для обнаружения аномалий
• ex8_movies.mat - набор данных обзора фильмов
• ex8_movieParams.mat - параметры, предоставляемые для отладки
• multivariateGaussian.m - вычисляет функцию плотности вероятности для гауссовского распределения.
• visualizeFit.m - 2D-график гауссовского распределения и набора данных
• checkCostFunction.m - проверка градиента для совместной фильтрации
• computeNumericGradient.m - численное вычисление градиентов
• fmincg.m - процедура минимизации функции (аналогично fminunc)
• loadMovieList.m - загружает список фильмов в массив ячеек
• movie_ids.txt - Список фильмов
• normalizeRatings.m - средняя нормализация для совместной фильтрации
• submit.m - скрипт отправки, который отправляет ваши решения на наши серверы
• * ratingGaussian.m - оценка параметров гауссовского распределения с диагональной ковариацией
матрица
• * selectThreshold.m - Найти порог для обнаружения аномалии
• * cofiCostFunc.m - реализовать функцию стоимости для совместной фильтрации
